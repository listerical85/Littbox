Littbox

An alternate lightbox, inspired by the Apple webshop.
Simply put, it starts and ends it's animation on the clicked image.
This gives the impression that the image is being enlarged on the fly.

Getting started

  1. Download the latest version of Littbox.
  2. Make sure you have jQuery loaded in your site. (If not, it is included on Github). <script src="js/jquery-1.10.2.min.js"></script>
  3. Find littbox.js in the js folder and include it in your site, after you have loaded jQuery <script src="js/littbox.js"></script>
  4. In the css folder you will find littbox.css include it in your site. Littbox uses one image for preloading, it assumes this image is located in a folder named images. The image is called loader.gif <link rel="stylesheet" href="css/littbox.css">
  5. Now you're ready to use Littbox, wrap your thumbnail image in a link, wrap them in a container (div/article/etc.) Add a title and subtitle, like the example below.
    6. <article>
    <a rel="littbox" href="images/litt4.jpg">
        <img src="images/litt4-small.jpg" alt="img-4">
    </a>
    <h3 class="littbox-title">Flower</h3>
    <p class="littbox-subtitle">Random image shot in<br/>The Netherlands<br></p>
</article>
  6. Make sure you add rel="littbox" to your link, and the littbox-title, littbox-subtitle classes to there proper elements.
  7. Enjoy Littbox!
